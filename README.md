# Proyecto de Análisis de Datos

Este repositorio contiene ejercicios y prácticas relacionadas con el análisis de datos realizados durante un curso de formación.

## Descripción

El objetivo de este proyecto es aplicar diferentes técnicas de análisis de datos, limpieza de información y visualización para obtener conclusiones útiles a partir de distintos conjuntos de datos.

## Contenido

En este repositorio se pueden encontrar:
- Ejercicios prácticos de análisis de datos
- Archivos de datos utilizados en las prácticas
- Scripts o herramientas empleadas para el análisis
- Resultados y visualizaciones obtenidas

## Herramientas utilizadas

- Excel / Google Sheets
- Python
- Power BI
- Otras herramientas de análisis de datos

## Estado del proyecto

Este proyecto forma parte de una actividad formativa y se irá actualizando a medida que avance el curso.

## Autor

Proyecto realizado como parte de un curso de análisis de datos.
